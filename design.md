# Main Wall Climbing Gym - Professional Design System

## Design Philosophy

### Color Palette
**Primary Colors:**
- **Electric Blue**: #0f42ec (Primary brand color, CTAs, links)
- **Sunset Yellow**: #fae404 (Accent color, highlights, energy)
- **Deep Navy**: #07090f (Text, headers, strong contrast)
- **Pure White**: #ffffff (Backgrounds, negative space)

**Supporting Colors:**
- **Light Blue**: #bfceff (Subtle backgrounds, hover states)
- **Light Yellow**: #fffbca (Accent backgrounds, highlights)
- **Medium Gray**: #6b7280 (Secondary text, borders)
- **Light Gray**: #f9fafb (Section backgrounds)

### Design Principles
- **No Contrasting Gradients**: Never blend yellow and blue directly
- **Solid Color Applications**: Use individual colors purposefully
- **High Contrast**: Ensure 4.5:1 minimum ratio for accessibility
- **Minimal Aesthetic**: Clean lines, generous whitespace, purposeful elements
- **Professional Polish**: Every detail considered for premium feel

### Typography
- **Display Font**: Inter Bold (700) - Large headings, hero text
- **Heading Font**: Inter SemiBold (600) - Section headers, navigation
- **Body Font**: Inter Regular (400) - Paragraph text, descriptions
- **Accent Font**: Inter Medium (500) - Buttons, labels, emphasis

## Visual Language

### Header Design
- **Transparent Background**: Subtle blur effect with backdrop-filter
- **Minimal Navigation**: Clean typography, no unnecessary elements
- **Sticky Behavior**: Smooth transition on scroll

### Component Styling

#### Buttons
- **Primary**: Solid electric blue background, white text
- **Secondary**: Solid yellow background, navy text
- **Ghost**: Transparent with blue border, blue text
- **Hover**: Subtle scale and shadow effects, no color morphing

#### Cards & Containers
- **Clean Backgrounds**: Pure white or light gray
- **Subtle Borders**: Light gray or blue accent
- **Minimal Shadows**: Soft, professional depth
- **Consistent Spacing**: Systematic padding and margins

#### Interactive Elements
- **Focus States**: Blue outline for accessibility
- **Hover Effects**: Subtle scale and elevation changes
- **Active States**: Pressed effect with reduced opacity

### Background Strategy
- **Solid Colors**: Consistent background throughout
- **Subtle Textures**: Minimal geometric patterns if needed
- **No Gradients**: Solid colors only for professional appearance
- **Section Differentiation**: Light gray backgrounds for content areas

## Implementation Guidelines

### Color Usage
- **Yellow**: Highlights, CTAs, energy elements, success states
- **Blue**: Primary actions, links, trust elements, interactive states
- **Navy**: Main text, headers, strong contrast elements
- **Light Colors**: Backgrounds, subtle accents, hover states

### Layout Principles
- **Grid System**: 12-column responsive grid
- **Whitespace**: Generous spacing for breathing room
- **Hierarchy**: Clear visual weight through size and color
- **Alignment**: Consistent left-align for text, centered for elements

### Animation Strategy
- **Subtle Motion**: 150-300ms transitions
- **Purposeful Effects**: Only when enhancing UX
- **Easing**: Smooth, natural curves
- **Performance**: Hardware-accelerated transforms only

This design system creates a cohesive, premium experience that positions Main Wall Climbing Gym as a high-end destination while maintaining excellent usability and accessibility standards.
